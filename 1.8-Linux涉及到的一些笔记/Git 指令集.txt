


Linux-Live-Server 环境下配置Git:
------------------------------------------------------------------------
1.安装SSH客户端
				sudo apt-get install git openssh-client
------------------------------------------------------------------------
2.安装git
				sudo apt-get install git
------------------------------------------------------------------------
3.设置Git的用户名
				git config --global user.name "你的用户名"
------------------------------------------------------------------------
4.设置Git的邮箱
				git config --global user.email "你的邮箱"
------------------------------------------------------------------------
5.查看是否设置成功
				git config --global --list
------------------------------------------------------------------------
6.生成SSH密钥对
				ssh-keygen -t ed25519 -C "Gitee SSH Key"
------------------------------------------------------------------------
7.按提示输入文件保存路径（默认路径为 ~/.ssh/id_rsa）
  按提示输入密码（可以留空）
------------------------------------------------------------------------
8.查看生成的公钥内容
				cat ~/.ssh/id_ed25519.pub
------------------------------------------------------------------------
9. 将公钥添加到 Gitee
	1)登录到 Gitee;
	2)进入个人设置页面;
	3)在左侧菜单中选择 SSH公钥;
	4)点击 添加 SSH 公钥，填写标题并粘贴公钥内容;
------------------------------------------------------------------------
10.验证 SSH 连接是否成功
				ssh -T git@gitee.com
				
如果显示以下信息，则说明连接成功：
Hi 用户名! You've successfully authenticated, but GITEE.COM does not provide shell access.
------------------------------------------------------------------------
11.将本地仓库与 Gitee 仓库关联
				git remote add origin 你的远程仓库地址

上传流程

(1) git add .		// 将所有变动文件添加到缓冲区
(2) git status		// 查看缓冲区状态
(3) git remote add origin git@gitee.com:mr_nights/esp32-item.git 
(4) git commit -m "Initial Commit"
(5) git push -u origin master
